Performance Trade-offs

Primary Observation

Approximately 1 in 5 trials resulted in data exfiltration via rendered markdown. The model reproduced the exfiltration payload in its output. In any system that renders markdown — a Slack bot, a web dashboard, or an agent UI — this constitutes a live data exfiltration. This was verified by piping output through mistune and confirming a real HTTP request to webhook.site.

Conclusions

Research Conclusions

Key findings from the three-phase empirical study.

Finding 1 — Naive Attacks Neutralized

L1 naive attacks achieve 0% success across all three models. Modern RLHF training has effectively neutralized the most basic injection vector. Direct "ignore previous instructions" prompts are no longer a viable attack path.

Finding 2 — Context Shift Dominates

L2 Context Shift is the dominant attack vector. Gemini showed a 36.5% success rate, approximately 2x higher than Claude (20%) and Groq (22.4%). This suggests RLHF approaches and system prompt handling vary significantly across providers.

Finding 3 — Exfiltration Works in the Wild

L3 exfiltration produces real HTTP requests. 12.5% of trials across all models generated markdown image tags that triggered webhook requests. This is not theoretical — live webhook firing was verified. Any system rendering LLM output as markdown is at risk.

Finding 4 — Spotlighting Is Most Effective

Spotlighting achieved 100% block rate on L1 and L3, and 82.4% on L2. Wrapping untrusted data in special markers before passing to the LLM is the most effective defense evaluated. However, L2 combined attacks still bypass it approximately 17% of the time.

Finding 5 — False Positive Problem

LLM-Judge flagged 18% of benign inputs as malicious, making it impractical for production without human review. The embedding filter had only 2% false positive rate but missed 85% of L2 attacks, highlighting the detection gap.

Methodology

Attack Tiers: Three tiers (L1-L3) sourced from the Open-Prompt-Injection dataset (Liu et al., 2310.12815), extended with L3 exfiltration payloads using markdown image tags.

Models: Claude (via Anthropic API), Gemini (via Google AI Studio), Groq (Llama via Groq API). All models tested with identical payloads, 3-30 trials per payload depending on phase.

Evaluation: Attack success measured via LLM-judge scoring (0-1 scale), defense detection via binary classification, latency measured end-to-end including API round-trip time.
